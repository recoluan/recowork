RecoWork · AI Work Harness

让 AI Agent按标准工作 Give AI agentsstandards to work from

RecoWork 是一套 AI Work Harness：把任务、专业标准、项目事实和验收方式固定下来，让每次协作都可检查、可续接。

问题不在 Agent 不够强

你不缺更聪明的 Agent，缺的是让它有标准可依地参与工作。You do not need a smarter agent.You need standards it can work from.

01

每次新对话，都重新解释背景。

目标、已有决定和约束散落在不同对话里。

02

AI 产出很多，却没有可信的当前版本。

草稿、讨论和结论混在一起，下一次无法按需接续。

03

部署了 Agent，却不知道交给它什么。

任务边界、完成标准和人工审查位置并不明确。

把一项真实工作固定成 Agent 可以持续推进的方式。

不要先配置一个什么都能做的 Agent。

两种使用环境

同一项工作，按持续性选择合适的入口。

01

本地可执行 Agent

适合长期项目、代码、复杂学习与持续工作。得到可维护的项目规范、工作空间、过程产物与验收方式。

02

Chat / Mobile

适合快速开始、临时任务和轻量对话。得到可复制的启动指令、执行协议、手动续接与迁移摘要。连续性依赖你保存并粘贴摘要。

当任务需要多人协作、长期知识沉淀或可审计过程时，从 Chat / Mobile 生成续接包，再迁移到本地可执行 Agent。

边界清楚，控制权在你

你控制 Agent，RecoWork 不替你运行它。

  • 不托管模型或 API Key。
  • 本地项目规范和产物保存在你选择的位置。
  • AI 的建议、产出和修改仍需要你审查。
  • 不承诺自动记忆、后台执行或无限自动化。
