Starter workflow templates

Start with three reusable AI work templates.

general

general-ai-workflow

Helps everyday users run tasks reliably in mobile or web AI apps with a role contract, task context, continuation memory, review, and reusable learning.

  • For ChatGPT Mobile / Claude Mobile / Kimi
  • Keeps context, quality standards, and proven methods reusable
project

project-engineering

Sets up AGENTS.md, a document standard, progressive retrieval, quality gates, and durable capture for AI-assisted projects.

  • For Codex, Cursor, and Claude Code
  • Gives a project an AI working protocol from day one
learning

learning-engineering

Turns learning into diagnosis, roadmaps, lessons, practice, projects, feedback, retrospectives, and durable knowledge capture.

  • For LangChain, coding, product, design, and more
  • Validates one learning unit at a time instead of dumping answers

How it works

Choose a workflow template, then choose where it will run. The CLI copies the template and applies the target output rules.

For builders

Templates and targets evolve independently.

templates/

Stores scenario templates such as project work, learning, or daily AI use.

locales/

Stores reusable working methods for a scenario, such as steps, principles, and checklists.

targets/

Stores reusable output rules for chat, project, workspace, and document targets.

examples/

Stores real scenarios so users can understand how to apply the pack.
