使用方式

不是运行一个工具，而是让 Agent 开始一项工作。This is not about running a tool.It is about helping an agent start real work.

先选择要完成的工作，再选择你与 AI 协作的环境。RecoWork 给出对应的开始方式、检查要求和续接边界。

五步开始

让 Agent 从第一次任务就有清楚的边界。

01选择真实工作

项目、学习、想法或网页设计规范。

02选择使用环境

本地可执行 Agent 或 Chat / Mobile。

03获得开始方式

复制聊天启动指令，或由本地 Agent 初始化。

04执行并检查

按任务协议交付，并按清单自检。

05保存续接信息

把当前事实留给下一次工作。

先选环境

本地可执行 Agent 与 Chat / Mobile，不是同一种工作方式。

01

本地可执行 Agent

适合长期项目、代码、复杂学习和持续工作。初始化项目规范、工作空间、过程产物和验收方式；在你选择的本地目录中维护。

02

Chat / Mobile

适合快速开始、临时任务和轻量对话。只提供启动指令、任务执行协议和续接/迁移摘要；不会创建本地目录，也不会自动保存过程。

Chat / Mobile 的连续性依赖你保存并粘贴续接摘要。任务变复杂、需要多人协作、长期知识沉淀或可审计过程时，使用迁移包转到本地可执行 Agent。

你想完成什么工作？你将在哪里和 AI 协作？
