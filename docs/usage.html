使用说明

从选择模板到生成文件，一次讲清楚。

RecoWork 用 template + target + locale 组装一套 AI 工作 harness。你先选择要解决的工作场景，再选择真实使用位置，最后选择输出语言。

选择初始化方式 查看 CLI 命令

四步使用

把选择过程拆成稳定决策。

01

选择模板

模板描述你要工程化的工作场景：日常 AI 使用、项目协作，或系统学习。

02

选择 target

Target 不是品牌名，而是具体使用位置，例如 codex-project、claude-chat 或 notion-workspace。

03

选择语言

Locale 控制用户可见的文件夹、文档和正文语言；AGENTS.md、CLAUDE.md 等约定文件名不会被翻译。

选择矩阵

先判断你在哪里使用 AI，再选择 target。

使用环境 推荐 target 生成内容
ChatGPT / Claude 网页或手机端 chatgpt-chat / claude-chat 可复制 Prompt、任务说明、续聊记忆
Codex / Cursor 本地项目 codex-project / cursor-project 项目规则、工作空间、知识库、质量检查
Claude Code 本地项目 claude-code-project CLAUDE.md 和 .claude/skills
Notion / 飞书文档协作 notion-workspace / feishu-doc 页面模板、工作区结构、复盘文档

CLI 命令

recowork CLI 负责把模板确定性地生成到目标位置。

01

list

查看当前可用的工作流模板，例如 project、general、learning。

npx recowork list
02

targets

查看可生成到哪些真实使用位置，例如 codex-project、chatgpt-chat、notion-workspace。

npx recowork targets
03

add

把一个模板按 target 和 locale 初始化到指定目录。

npx recowork add project --target codex-project --locale zh .
04

upgrade

先检查模板更新和工作空间建议；只更新未改动的方法和 target 文件。

npx recowork upgrade --check .
# 查看模板
npx recowork list

# 查看 targets
npx recowork targets

# 初始化项目工作流到当前目录
npx recowork add project --target codex-project --locale zh .

# 初始化聊天工作流到新目录
npx recowork add general --target chatgpt-chat --locale zh ./my-ai-workflow

# 初始化学习工作流到 Notion 文档工作区
npx recowork add learning --target notion-workspace --locale zh ./my-learning-workflow

# 只读检查已有工作流的更新建议
npx recowork upgrade --check .

# 安全更新未改动的工作方法和 target 文件
npx recowork upgrade --apply .

# 仅补齐新版新增且当前缺失的工作空间文件
npx recowork upgrade --apply --scope workspace --add-missing .

怎么用

选择一个工作流模板、运行位置和语言。CLI 会复制模板并套用 target 输出规则。

初始化方式

同一个配置，可以通过两条路径落地。

让 AI 帮你初始化

适合 Codex、Cursor、Claude Code 等能运行命令的本地智能体。Prompt 会要求 AI 先检查 Node.js 和 npm，再运行 npx recowork。

纯聊天 / 手机端

适合没有终端和文件系统的环境。RecoWork 会生成可复制的聊天启动 Prompt，而不会要求用户安装 Node.js 或创建本地文件。

验收结果

初始化完成后，检查这些东西是否真的生成。

规则文件

项目 target 应有 AGENTS.md、CLAUDE.md 或工具对应的规则文件；聊天 target 不应该生成项目工具目录。

工作方法

检查角色设定、文档规范、工作流程、质量检查是否与所选模板和语言一致。

工作空间

项目和学习模板应生成可持续维护的工作空间或学习空间，并包含 index.md 导航。
