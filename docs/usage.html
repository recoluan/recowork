使用说明

从选择模板到生成文件，一次讲清楚。

RecoWork 分为两层：Chat / 手机端提供即时可用的轻量对话入口；Codex、Claude Code、Cursor 等本地可执行 Agent 承载完整、可持续的工程化工作流。

选择初始化方式 查看 CLI 命令

三步使用

把选择过程拆成稳定决策。

01

选择模板

模板描述你要工程化的工作场景：日常 AI 使用、想法探索、项目协作、系统学习或网页设计。

02

选择 target

选择聊天 / 手机端或电脑端 AI 助手，两者的能力边界不同。

03

选择语言

Locale 控制用户可见的文件夹、文档和正文语言；AGENTS.md 等约定文件名不会被翻译。

选择矩阵

先判断你在哪里使用 AI，再选择 target。

使用环境 推荐 target 生成内容
任意网页、App 或手机聊天 chat-mobile 启动指令、任务执行协议、手动保存的续接/迁移摘要；不创建本地工作空间
电脑端 AI 助手 local-agent-project 可执行命令、读写项目文件；创建或安全整合根目录 AGENTS.md，并提供状态检查与安全升级

CLI 命令

recowork CLI 负责把模板确定性地生成到目标位置。

01

list

查看当前可用的工作流模板，例如 project、general、learning、idea、web-design。

npx recowork list
02

targets

查看两种使用环境：chat-mobile 与 local-agent-project。

npx recowork targets
03

add

把一个模板按 target 和 locale 初始化到指定目录。

npx recowork add project --target local-agent-project --locale zh .
04

upgrade

先检查模板更新和工作空间建议；只更新未改动的方法和 target 文件。

npx recowork upgrade --check .
# 查看模板
npx recowork list

# 查看 targets
npx recowork targets

# 初始化项目工作流到当前目录
npx recowork add project --target local-agent-project --locale zh .

# 初始化学习工作流到电脑端 AI 助手
npx recowork add learning --target local-agent-project --locale zh ./my-learning-workflow

# 只读检查已有工作流的更新建议
npx recowork upgrade --check .

# 安全更新未改动的工作方法和 target 文件
npx recowork upgrade --apply .

# 仅补齐新版新增且当前缺失的工作空间文件
npx recowork upgrade --apply --scope workspace --add-missing .

迁移出口

聊天用于快速开始，本地 Agent 用于长期推进。

Chat / 手机端

复制启动指令，在对话内完成澄清、确认和自检。每次重要回复保存续接摘要；平台不会自动保存中间产物或维护版本。

迁移到本地 Agent

当需要长期沉淀、多人协作、可审计过程或安全升级时，将项目简报、当前决策、未决问题和下一步粘贴到具备命令执行能力的本地 Agent，初始化完整工作空间。

开始初始化

选择配置，再选择你习惯的初始化方式。

怎么用

选择一个工作流模板、运行位置和语言。CLI 会复制模板并套用 target 输出规则。

初始化方式

同一个配置，可以通过两条路径落地。

让 AI 帮你初始化

适合能运行命令的本地 Agent。Prompt 会要求 AI 先检查 Node.js 和 npm，再运行 npx recowork。

纯聊天 / 手机端

适合没有终端和文件系统的环境。RecoWork 会生成可复制的聊天启动 Prompt，而不会要求用户安装 Node.js 或创建本地文件。

验收结果

初始化完成后，检查这些东西是否真的生成。

规则文件

本地 target 应创建根目录 AGENTS.md，或在已有文件中加入带 RecoWork 标记的独立区块；聊天 target 不应生成项目工具目录。

工作方法

检查角色设定、文档规范、工作流程、质量检查是否与所选模板和语言一致。

工作空间

项目和学习模板应生成可持续维护的工作空间或学习空间，并包含 index.md 导航。

RecoWork 面向真实 AI 工作的开源工作流模板。
