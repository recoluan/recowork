工作案例

先把一项常做的工作交给 Agent 持续完成。Start with one jobyour agent can keep completing.

这些不是抽象的能力标签，而是你能观察到任务、事实、检查和下一步的实际工作。

01 · 项目工程化

把项目从“讨论很多”推进到“当前事实清楚”。

适合谁
要推进产品、技术或业务项目的人。
首次任务
澄清目标、范围、约束和未决问题，等待关键方向确认。
留下什么
项目简报、方案、决策、过程留痕和验收记录。
何时迁移
当需要长期推进、多人协作或审计过程时，使用本地可执行 Agent。
开始项目工作
02 · 日常 AI 工作流

让重复任务不必在每次对话都从零解释。

适合谁
经常用聊天助手分析、整理、写作或规划的人。
首次任务
明确一次任务的输入、输出、自检方式和续接摘要。
留下什么
任务协议、当前结论、未决问题和下一步。
当前状态
RecoWork 暂未提供独立的日常任务模板；可先用 Chat / Mobile 的轻量对话方式验证自己的重复任务，再选择项目、学习或想法工作流。
选择使用环境
03 · 学习工程化

把学习从内容堆积推进到能力验证。

适合谁
需要系统学习一门技术、专业知识或实践能力的人。
首次任务
诊断基础、目标和时间约束，先确认学习约定。
留下什么
学习简报、路线、进度、练习、项目实践和复盘。
何时迁移
当课程跨多个单元、要积累练习证据时，使用本地可执行 Agent。
开始学习工作
04 · 想法工程化

把灵感推进成有证据、有下一步的方向。

适合谁
正在探索产品、业务、内容或研究方向的人。
首次任务
澄清问题、用户、约束与成功信号，再开始发散。
留下什么
问题定义、候选方向、假设、验证结果与下一步决策。
何时迁移
当探索变成持续验证或需要与团队共享事实时，使用本地可执行 Agent。
开始想法工作
