# 项目工程化工作流包

这个包用于在项目刚开始时建立 AI 工作协议。它先澄清并等待用户确认项目约定，再生成方案、计划或实施改动，以减少意图偏差、上下文漂移和无沉淀的重复劳动。

适合场景：

- 用具备命令执行能力的本地 Agent 做项目。
- 希望每次改动都有计划、验收和权威记录更新。
- 希望跨本地 Agent 复用同一套工作方法。

核心交付：

- `AGENTS.md`
- `工作方法/` 工作规则和方法
- `工作空间/` 项目工作空间
- `index.md` 分层索引
- `工作方法/角色设定.md`
- `工作方法/文档规范.md`
- `工作方法/产物保鲜规范.md`
- `项目简报.md`
- `待确认问题.md`
- 自审和验收规则
