# 示例：LangChain TypeScript 学习项目

学习者有 Web 前端和 Node.js 基础，希望通过 TypeScript 掌握 LangChain，并完成一个带来源引用的知识库助手。

1. 在 `学习简报.md` 记录基础、目标、每周时间和项目完成标准。
2. 在 `课程路线.md` 按模型与消息、Prompt、文档处理、检索、RAG、Agent、评估等依赖拆分章节。
3. 每章创建 `NN-主题/`，保留讲义、练习、参考实现、实际运行命令和个人理解。
4. 用贯穿式知识库助手逐步串联课程内容，而不是只做孤立示例。
5. 将 API 变化、错误原因、数据流理解和可复用模式写入 `04-问题与复盘/` 或 `05-知识沉淀/`。
