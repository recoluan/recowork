# 学习工程化工作流包

这个包把“想学一个主题”变成可持续的课程与实践系统。它先完成学习诊断并等待学习者确认学习约定，再通过章节、练习、项目和复盘建立真实能力，而不是一次性塞满知识。

适合场景：

- 系统学习编程、AI、产品、设计或任意有实践要求的主题。
- 希望 AI 像严谨导师一样逐章教学，而非直接倾倒答案。
- 希望保留每章练习、运行验证、问题、个人理解和项目成果。

核心产物：

- `工作方法/`：学习导师角色、教学方法、课程单元、评估规则和产物保鲜规范。
- `学习空间/`：学习简报、课程路线、进度、课程、实践与知识沉淀。
- `学习简报.md`、`课程路线.md`、`学习进度.md`。
- 可复用的课程单元、项目实践与复盘模板。
