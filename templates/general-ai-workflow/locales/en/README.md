# General AI Workflow Template

This template is for everyday AI users. It turns one task into a lightweight workflow that can clarify, continue, review, and improve over time. It does not require knowledge of skills, agents, or project directories.

Good fit:

- You use ChatGPT, Claude, Kimi, Doubao, or similar AI in a mobile or web chat.
- You work on writing, summaries, planning, analysis, communication, or information organization.
- You want AI to clarify important questions before producing an answer.
- You want the next conversation to restore context and reuse what worked.

Core outputs:

- `methods/`: stable working rules, role contract, and review standards.
- `workspace/`: context, outputs, traces, and retrospectives for the current task.
- `methods/role-contract.md`: the AI role, boundaries, and iteration rules.
- `task-brief.md`, `open-questions.md`, and a continuation memory template.
- initialization and task prompts for the selected target.
