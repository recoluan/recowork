# Artifact Freshness Standard

The idea brief, priority directions, validation conclusions, and next-step decisions are default sources; brainstorm traces are for history. Use `Draft`, `In review`, `Current`, `Superseded`, and `Archived`; one topic has one Current version.

At the end of each exploration, consolidate valid conclusions, mark traces absorbed, pending, or trace-only, and update Current and Historical index entries. Never delete user content automatically; archival or deletion requires confirmation.
