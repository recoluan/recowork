# Idea Exploration And Validation Template

Use this template to brainstorm a fuzzy idea with AI. It explores possibilities, separates directions from facts and assumptions, then uses small validation steps to converge on what is worth pursuing instead of treating a long idea list as a conclusion.

Core outputs:

- `methods/`: exploration role, document standard, divergence and convergence, hypothesis validation, and artifact freshness.
- `idea-space/`: brief, directions, traces, validation, and decisions.
- A confirmed direction and project agreement that can move into `project-engineering`.
