# Project Start

1. Clarify project goal, audience, constraints, and success criteria.
2. Identify assumptions and questions requiring confirmation.
3. Propose project structure and knowledge files.
4. Produce a first-phase plan.
5. Wait for confirmation before broad implementation when the request is ambiguous.
