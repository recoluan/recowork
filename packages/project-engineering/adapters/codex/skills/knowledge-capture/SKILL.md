# Knowledge Capture

After meaningful work:

1. Add stable rules to `knowledge/principles.md`.
2. Add decisions and reasons to `knowledge/decisions.md`.
3. Add reusable methods to `knowledge/patterns.md`.
4. Add pitfalls to `knowledge/mistakes.md`.
5. Add progress notes to `knowledge/project-log.md`.
