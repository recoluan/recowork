# 更新日志

RecoWork 的重要变更记录在此。项目遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 与 [语义化版本](https://semver.org/lang/zh-CN/)。

## [未发布]

### 新增

- 新增 `web-design-standard`：面向产品型网页的可复用设计规范；本地 Agent 交付一份本地化规范文件，聊天 / 手机端交付一份可独立使用的响应式网页大 Prompt。

### 变更

- 下线 `general-ai-workflow`：日常任务范围没有形成足够明确、可复用的工作流价值。新初始化请改选想法、项目或学习工作流；已有 general 工作流文件保持不动，运行 `rw status` 或 `rw upgrade` 会获得只读迁移指引。
- 本地初始化遇到已有根目录 `AGENTS.md` 时，会安全写入带标记边界的 RecoWork 区块；升级时会保留用户修改或删除过的受管区块。
- 工作流角色设定现在要求客观识别风险、矛盾、薄弱假设与可信替代方案，不得一味迎合。
- 文档站在能力目录和初始化流程中都将工作流模板与设计规范分开，并在发布摘要中直接列出每项待发布能力。
- 文档站试用在线加载的 Noto Sans SC 与 JetBrains Mono，统一中英文与代码字体；暂不写入本地字体资产。
- 站点大标题改用平衡换行并使用完整区块宽度，避免字体调整后出现生硬的词组截断。

### 兼容性

- 已有根目录 `AGENTS.md` 在 RecoWork 受管区块之外的内容保持不变；已有工作流不会被自动修改。
- 已有 `general-ai-workflow` 文件保持不动；请通过 `rw status` 或 `rw upgrade` 的迁移指引，在单独目录初始化受支持的工作流。

### 破坏性变更

- 不再支持初始化、列出或查看 `general-ai-workflow` 及其 `general`、`task`、`daily` 别名。

## [1.0.0] - 2026-07-21

### 新增

- 新增 `chat-mobile` target，交付启动指令、任务执行协议和手动保存的续接/迁移摘要。
- 新增 `local-agent-project` target，交付工具中立的 `AGENTS.md`、工作方法、持续工作空间与安全升级能力。
- 为旧 Chat 工作流增加迁移指引：在新目录初始化本地工作流，旧文件保持不动。

### 变更

- 将品牌专属 target 收敛为两种用户环境：聊天 / 手机端与电脑端 AI 助手。
- 将长期知识收口到工作空间中的权威文档与索引，不再生成独立知识库目录。
- 当目标目录已有 `rw-manifest.json` 时，`rw add` 会拒绝执行，防止重新初始化覆盖既有工作流。
- 同步更新模板、CLI Prompt、中英文 README、规格与文档站。

### 移除

- 移除品牌专属 target 输出、原生 skills、平台配置目录，以及 Notion/飞书文档导出。

### 兼容性

- 旧 target 名称保留为 CLI 别名，映射至 `chat-mobile` 或 `local-agent-project`。
- 已有工作空间保持不动；旧 Chat 工作流会获得指向独立本地目录的迁移命令。

### 破坏性变更

- 不再生成品牌专属路径和原生 skill 输出。完整本地工作流请使用 `local-agent-project`，轻量对话请使用 `chat-mobile`。

## [0.4.0] - 2026-07-20

### 新增

- 学习工作流在生成或变更课程路线、章节、练习计划或项目方案前，必须先获得学习约定的明确确认。
- 项目工作流在生成或变更完整方案、计划或实施改动前，必须先获得项目约定的明确确认。
- 中英文角色设定、工作方法、聊天提示词和项目指令均包含确认门禁。

### 变更

- `learning-engineering` 与 `project-engineering` 模板版本升级为 `1.1.0`。
- 发布到 npm 的包现在展示仓库、官网和问题反馈链接；内置 CLI 说明增加引导式使用、源码、发布记录和 AI 初始化入口。

### 兼容性

- 已有工作流不会自动改动。运行 `rw upgrade --plan .` 查看可用的方法和 target 更新；仅未被用户修改的文件可安全应用。
- 新初始化只会在大范围学习或项目设计工作前暂停等待明确确认；已确认的局部任务无需重复确认。

## [0.3.0] - 2026-07-12

### 新增

- 支持按中英文 locale 输出 target 的用户可见路径和静态内容。
- 为持续工作流模板加入文档规范、索引优先读取和索引维护机制。
- 为贡献者和 AI 智能体加入 Conventional Commits 提交规范。
- 加入保守的工作流升级顾问：版本化清单、只读检查、安全更新和工作空间升级报告。

### 变更

- 中文初始化现在生成 `知识库/`、`示例/` 等用户可见中文路径，同时保留工具约定文件。
- locale 切换时，会安全清理上一语言遗留且可替换的生成文件。
- 升级时工作空间明确归用户所有：已有工作空间内容不会被覆盖、移动、删除或恢复。
- 升级报告写入 `.recowork/upgrade-reports/`，位于用户工作空间之外。

### 兼容性

- 已有工作流不会被自动改动；旧版清单需先执行 `rw upgrade --adopt .`。
- 新增工作空间模板需显式使用 `--scope workspace --add-missing`；已有工作空间文件保持不变。
